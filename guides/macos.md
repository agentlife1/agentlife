# AgentLife — macOS Install Guide

## Prerequisites

- macOS 13 (Ventura) or later
- [Homebrew](https://brew.sh) recommended
- Python 3.10+ (`brew install python`)

## Step 1: Install Hermes Agent

Follow the [Hermes Agent install guide](https://hermes-agent.nousresearch.com/docs) for macOS.

## Step 2: Install AgentLife

```bash
curl -sfSL https://agentlife.io/install.sh | bash
```

## Step 3: Configure

```bash
agentlife setup
```

Select your persona(s) and use cases.

## Step 4: Verify

```bash
agentlife verify
```
